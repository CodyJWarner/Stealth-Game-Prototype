# Stealth-Game-Prototype
A third-person stealth prototype built in Unreal Engine 5. The project focuses on stealth mechanics, enemy detection, player objectives, and win/loss conditions.
<br>

*MAIN MENU GIF PLACE HOLDER*


The main objective is to sneak through the level, collect the scroll, and reach the exit without being caught. The prototype includes security cameras, trip wires, and a patrolling guard that can detect the player and trigger a loss condition.
I also added UI elements such as a timer, win screen, loss screen, main menu, controls screen, and credits screen to make the project feel more complete as a playable prototype.
